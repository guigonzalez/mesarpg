# Attached Assets

Este diretório contém recursos e documentação importantes para o projeto MesaRPG.

## 📁 Estrutura

- **`docs/`** - Documentação do projeto
  - `PRD_MesaRPG.md` - Documento de Requisitos do Produto
- **`data/`** - Dados estruturados
  - `Fichas_Simplificadas_por_Sistema.csv` - Templates de fichas por sistema RPG
  - `Campos_com_Subcampos_por_Sistema.csv` - Campos detalhados por sistema
- **`images/`** - Imagens e screenshots
  - Screenshots de desenvolvimento
  - Elementos visuais de referência

## 📋 Conteúdo

### Documentação
- **PRD_MesaRPG.md**: Especificações detalhadas do produto, requisitos, análise competitiva e roadmap

### Dados
- **CSV Files**: Templates e estruturas de dados para diferentes sistemas de RPG (D&D 5e, Tormenta20, Call of Cthulhu, etc.)

### Imagens
- Screenshots de desenvolvimento e elementos visuais de referência

## 🎯 Uso

Estes arquivos servem como referência para:
- Desenvolvimento de funcionalidades
- Implementação de templates de personagem
- Documentação do projeto
- Referência visual durante o desenvolvimento
